# Handyman Services Starter

A Vercel-ready Next.js website template for handyman, property maintenance, and local trades businesses that need a clean lead-generation website with a premium feel.

## What This Template Is For

This starter is designed for:

- independent handyman businesses
- landlord and property maintenance services
- general trades businesses
- local home-repair companies

It includes a production-ready marketing structure focused on:

- trust
- local SEO
- mobile-first enquiries
- quote requests

## Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4
- Vercel-ready deployment

## Pages Included

- Home
- Services
- Quote

## Features

- multi-page local-service layout
- clear call and quote CTAs
- local SEO-friendly structure
- premium but practical design direction
- service-area and trust-signal sections
- built-in footer attribution

## Local SEO Notes

This starter is structured to make expansion easier:

- add individual service pages
- add location pages
- add FAQ content blocks
- add testimonials and review schema
- add stronger internal linking over time

## Getting Started

1. Install dependencies:

```bash
pnpm install
```

2. Run locally:

```bash
pnpm dev
```

3. Build for production:

```bash
pnpm build
```

## Deploy On Vercel

- import this repository into Vercel
- set `NEXT_PUBLIC_SITE_URL`
- deploy

If you want a one-click Vercel button, add one after publishing this repo to your own GitHub organization so the repository URL points at your canonical source.

## Environment Variables

See `.env.example`.

Required:

- `NEXT_PUBLIC_SITE_URL`

## Forking And Rebranding

Before launching publicly, update:

- site name, copy, phone numbers, and email addresses
- metadata in `app/layout.tsx`
- any placeholder service areas and trust content
- branding assets, colors, and imagery
